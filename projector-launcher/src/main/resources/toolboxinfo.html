Warning

We will remove the Projector desktop app from JetBrains Toolbox in roughly two weeks.

New versions of the app will be available on GitHub Releases.

The primary remote development tool for JetBrains IDEs is Gateway. Gateway gives you the ability to work with minimal network latency, and setting up the infrastructure and administering it is simpler than it is with Projector.

Projector is a special solution for those who can't use Gateway. Projector has much higher network latency and is more difficult to configure, but it is the recommended solution if you want to access your IDEs from web browsers (including mobile browsers).

If you're not sure which solution you should choose, please consider using Gateway.
